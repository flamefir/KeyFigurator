# domains/ — loops

Each subfolder is one **loop**: a charter, a cadence, optional metrics. A domain folder holds
only its `README.md` (live state) + optional machinery (`metrics/*.jsonl`). It LINKS artifacts
in `signals/` and `docs/`; it never contains them. To-dos live inline in `## Backlog` until
they earn a `task` kind. See `ARCHITECTURE.md`.
